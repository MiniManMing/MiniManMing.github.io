利用itchat玩转微信(环境的搭建)

发表于 2017-12-13 | 分类于 Teach | | 阅读次数:

前言

我的初衷

我感觉有时需要静下来做事的时候，微信总事会勾起我要看手机的欲望（个人定力差）微信的联系人和群挺多的有时会分散我的注意力。所以我建立了一个微信小号，目的是当有人找我有事时发微信给大号时程序可以把重要的消息转发到我的小号。（是不是很奇葩，估计有人会说你Y是不是有病是不是在炫技，微信明明有屏蔽功能啊！）其实我的看法是学习吗？就是要用一些技术去满足一下自己的奇葩欲望。最终目的还是学会一样技术吗！

项目所需

Language:Python 3.6.3
library:itchat
IDE:PyCharm 2017.2.4

搭建环境

注意：我这里在开发时用的是Windows10部署使用的是云主机上的Linux所以我会写出两种（仅供参考）

Windows上环境的搭建

  • 安装Python3.6.3 Link （安装直接狂点下一步）
  • 测试Python环境 Win+R 输入cmd然后输入python会出现Python Console
  • 安装PyCharm Link
  • PyCharm怎样和谐 Link C:\Windows\System32\drivers\etc hosts文件里上0.0.0.0 account.jetbrains.com
  • 安装itchat Win+R 输入cmd输入 pip install itchat

Linux上环境的搭建

  • 安装Python环境
  • 修改系统默认编译版本为Python3.6
  • 升级pip版本
  • 安装itchat
  • 未完待续 12月14日 更新。
